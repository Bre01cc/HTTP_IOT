## Sobre
Projeto de IoT utilizando os protocolos HTTP e HTTPS para realização de requisições, com exibição dos resultados por meio de LEDs: verde para indicar sucesso e vermelho para indicar erro.

Disciplina ministrada pela professora [Yuri Komuta](https://www.linkedin.com/in/yurikomuta/).
